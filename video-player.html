Understanding the Video State Machine

📡 loadstart Browser begins loading the video metadata
loadeddata First frame of video is loaded and ready
▶ play play() method is called - request to start
playing Playback actually starts (buffer ready) - distinct from 'play'
⏸ pause Playback is paused by user or system
waiting Buffering - playback stalled while loading more data
ended Video playback completes
error An error occurred during loading or playback
Poster Image: A poster image is displayed before the video loads and while loading, providing visual context to users.
