 Project Overview
EduStream is a comprehensive multimedia learning platform demonstrating:
- HTML5 media integration with custom controls
- Web Audio API synthesis with ADSR envelopes
- SVG animation and CSS keyframe techniques
- Real-time camera capture with canvas filters
- Component-based architecture with pure functions

 Pages Structure
| Page      | URL                       | Key Features |

| Audio Lab | audio-player.html       | Custom audio player, Web Audio synth, keyboard shortcuts |

 Technologies Used
- HTML5 (semantic elements, picture, video, audio)
- CSS3 (Grid, Flexbox, animations, transitions)
- JavaScript (ES6+)
- Web Audio API
- MediaDevices API (getUserMedia)
